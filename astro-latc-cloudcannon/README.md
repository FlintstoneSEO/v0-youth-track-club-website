# Lansing Area Track Club Astro Site

CloudCannon-friendly Astro rebuild of the Lansing Area Track Club website.

## Commands

```sh
npm install
npm run dev
npm run build
```

## Content

- `src/content/events`
- `src/content/announcements`
- `src/content/faqs`
- `src/content/run-tha-city-events`
- `src/content/sponsors`
- `src/data/*.json` for shared settings, navigation, footer, and practice schedule

Forms in `src/pages/contact.astro` use placeholder static form actions. Replace those URLs with the preferred form provider before launch.
